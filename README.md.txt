# Python Parallel Text Handling Processor

## Milestone 2 - Text Breaker and Loader

This project reads a large text file, splits it into smaller chunks,
and processes them in parallel using Python multithreading.

### Features
- Reads text files
- Splits text into chunks
- Processes chunks concurrently
- Applies regex-based word filtering
- Calculates total word count

### How to Run
1. Place text file inside data folder
2. Run: python main.py

### Technologies Used
- Python
- concurrent.futures
- Regular Expressions
